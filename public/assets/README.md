# Assets Folder

Dieser Ordner ist für statische Assets wie Bilder, Videos und andere Medien gedacht.

## Struktur

- `images/` - Für Fotos der Kapelle, Kunstwerke, etc.
- `icons/` - Für zusätzliche Icons
- `documents/` - Für PDFs oder andere Dokumente

## Verwendung

Platzieren Sie hier Ihre Bilder und referenzieren Sie sie im HTML mit `/assets/images/dateiname.jpg`
